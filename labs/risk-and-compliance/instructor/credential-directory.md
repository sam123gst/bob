# 🔐 Credential Directory

As instructors of the watsonx.governance lab, you should provide user credentials to each of your attendees in the format you see best fit. One common way to do this is to provide a CSV of credentials for the attendees to keep.

This README contains some quick template instructions that the Instructors may find useful to share with attendees.
---

## Access Instructions

1. **Login to IBM Cloud**
   - Navigate to the *IBM Cloud Authorization* Link above.
   - Enter your student email (e.g., `student0@techzone.ibm.com`)
   - Enter your assigned password

2. **Access Services**
   - Navigate to **Resource List > AI / Machine Learning**
   - Launch the **OpenPages** instance
   - If you receive an authorization error, add `/app/jspview/react/grc/dashboard/Home` to the end of the URL

3. **Verify Your Role**
   - After logging in, check your Profile
   - Ensure it is set as **Watsonx governance MRG Master**
   - This is required to access all governance features

---

## Important Notes

🔐 **Profile Setup:**
- Make sure your profile is set to **Watsonx governance MRG Master**
- This is critical for accessing governance features
- Contact the instructor if your profile is not configured correctly
